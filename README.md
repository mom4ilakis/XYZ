# XYZ

Developed with python 3.10

Setup:
 * Create a virtual env ```python -m venv venv```
 * Activate the virtual env
 * Install the requirements ```pip install -r requirements.txt```
 * Run ```python models.py``` to init the db
 * To calculate the distance to the nearest store for a giver city ```python main.py -c Some_City_Name```
  
  The result will be saved in ```task.db``` in the ```nearestshop``` table
